## Description:

This is a MERN application that allows users to search for books on the Google Books API and save them to a database. 

Technologies used:
<ul>
<li>React </li>
<li> MongoDB Atlas</li>
<li> Mongoose</li>
<li>Node </li>
<li>Express </li>
<li>Axios </li>
